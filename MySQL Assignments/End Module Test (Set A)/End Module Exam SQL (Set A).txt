Q4. Write a query to count the number of employees in each department from the 'Employees' table.

mysql->select dept_name, count(emp_id)
     ->from Employees
     ->group by dept_name;

Q5. Fetch the department-wise average salary from the 'Employees' table, only for departments where the average salary is greater than 50,000.

mysql->select dept_name, avg(salary)
     ->from Employees
     ->group by dept_name
     ->having salary>50,000;

17. Question:

• Books with columns: book_id, author_id, title, price. 
• Sales with columns: sale_id, book_id, sale_date, quantity.
